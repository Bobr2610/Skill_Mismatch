# 1. Введение

## 1.1. Актуальность исследования

В условиях стремительной цифровой трансформации мировой экономики проблема **несоответствия компетенций** (Skill Mismatch) становится одним из ключевых барьеров для устойчивого экономического роста. По данным Международной организации труда (ILO), от 25 до 45% работников в развитых странах занимают должности, не соответствующие их реальным навыкам — они либо переквалифицированы, либо недоквалифицированы относительно требований рабочего места.

Параллельно с этим действует **Принцип Питера** — организационный феномен, при котором сотрудники продвигаются по карьерной лестнице до тех пор, пока не достигнут уровня собственной некомпетентности. Этот эффект системно усиливает skill mismatch внутри организаций: вчерашний эффективный программист становится неэффективным менеджером, а его позицию занимает сотрудник, ещё не готовый к задачам такого уровня.

Совокупность этих явлений порождает **парадокс современного найма**: высокооплачиваемые специалисты вносят непропорционально малый вклад в конечный продукт, тогда как низкооплачиваемые работники критически важны для операционной деятельности. Традиционные системы оценки персонала, основанные на субъективных интервью, самоотчётах и формальных аттестациях, не способны объективно выявить и исправить эти диспропорции.

## 1.2. Цель и задачи

**Цель работы** — исследовать социологические и экономические аспекты проблемы Skill Mismatch и Принципа Питера, а также предложить интеллектуальную систему объективной оценки компетенций на основе анализа реальных рабочих артефактов.

**Задачи:**

1. Определить сущность и масштаб проблемы Skill Mismatch на основе данных ILO, OECD и академических исследований
2. Проанализировать механизм действия Принципа Питера и его влияние на организационную эффективность
3. Оценить экономические потери от неоптимального распределения человеческого капитала
4. Предложить математическую модель объективной оценки компетенций специалистов
5. Описать платформу DevMetrics как практическую реализацию предложенного подхода

## 1.3. Объект и предмет исследования

- **Объект** — процессы найма, оценки и продвижения специалистов в ИТ-индустрии
- **Предмет** — закономерности формирования skill mismatch и методы его выявления на основе автоматизированного анализа рабочих артефактов

## 1.4. Методология

Исследование основывается на:

- **Статистическом анализе** данных ILO и OECD по 37 странам
- **Социологическом анализе** явления Принципа Питера
- **Математическом моделировании** системы грейдирования с функцией затухания Грина
- **Программной реализации** платформы DevMetrics с применением ИИ (LLM) для анализа git-коммитов и метрик кода
